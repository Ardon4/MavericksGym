# Maverick's Gym App

A personal fitness tracker built in Python to log exercises, meals, sleep, and track overall progress.

# Features

- Log exercises and track reps/weights
- Track meals with protein, carbs, fats, and vitamins
- Log sleep and monitor sleep quality
- View summaries and progress reports
- AI Coach feedback (placeholder for ChatGPT integration)
- Reward system with motivational graphics
